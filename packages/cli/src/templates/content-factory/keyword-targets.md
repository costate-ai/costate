## Meta tag checklist

SEO agent verifies each published blog post has:

- `<title>` — 50–60 chars, primary keyword in first half
- `<meta description>` — 150–160 chars, primary keyword once, reads as a sentence
- `<h1>` — one per page, contains primary keyword
- At least one `<h2>` with a supporting keyword
- Canonical URL set
- Open Graph image 1200×630 with title overlay
- Alt text on every image, ≥ 5 words, includes a keyword where honest
